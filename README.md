# ROSNodeJs

```
1. git clone https://github.com/Hsu-Heng/ROSNodeJs.git
2. cd ROSNodeJs
3. Change NGINX_HOST which in docker-compose.yml line16
4. docker-compose up
5. Use ctrl+c to exit 
6. View the website in Explore which ip is NGINX_HOST
7. In next time, use "docker-compose start"  to start the container
8. Use docker-compose stop to stop container
```
